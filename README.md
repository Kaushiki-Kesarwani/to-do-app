# To-Do App

A simple and responsive To-Do app built with React. Keep track of your tasks, mark them as completed, and manage your day efficiently.

---

## 🌐 Live Demo

Check out the live app here: [To-Do App Demo](https://to-do-app-theta-indol.vercel.app)

---

## 🛠 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive design for mobile and desktop
- Clean and simple user interface

---

## 💻 Tech Stack

- **Frontend:** React
- **Deployment:** Vercel
- **Styling:** CSS / Tailwind CSS (if used)
- **State Management:** useState, useEffect (React Hooks)
